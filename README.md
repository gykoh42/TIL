## 🛠️ 작성 규칙

1. 매일 최소 한 개 이상 학습한 내용을 기록합니다.
2. 짧고 명확하게, 내가 나중에 봐도 이해할 수 있도록 작성합니다.
3. 커밋 메시지는 다음과 같은 형식을 따릅니다:
   ```
   [카테고리] 내용 요약
   예) [Java] 인터페이스와 추상 클래스 차이 정리
   ```
